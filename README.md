#  Weather Data Toolkit
### Client-Server Weather Lookup • Data Visualization • Meteostat Bonus

---

## Project Overview

This project is based on a university microproject with three parts:

1. A TCP client-server system that queries weather data from a local CSV file.
2. Weather diagrams created with Pandas and Matplotlib.
3. A bonus version that downloads weather data with Meteostat for Cologne/Bonn airport.

The assignment uses the local file `airport-cgn.csv` and focuses on weather values such as temperature, precipitation, pressure, and sunshine duration.
